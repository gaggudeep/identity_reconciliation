# Contact Management API

## Dependencies
```
- NodeJS (20)
- npm
- Postgres
- Docker (Optional) 
```

## Test
```sh
npm run test
```

## Build
```sh
npm install
```

### Run
- Ensure Postgres is running
```sh
npm run start
```

### Production build
```sh
npm run build
```
